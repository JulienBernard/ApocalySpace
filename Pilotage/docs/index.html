Informations

Author: Julien Bernard
Language: French

ApocalySpace est un jeu en ligne jouable gratuitement et directement sur votre navigateur web mêlant stratégie et gestion dans l'espace.

Version

Lastest version: 1.6.12
Stability of version: CLOSED BETA

IDE: Eclispe for Docs, Notepad++ for PHP/HTML and CSS
Design Pattern: MVC
Principal programmation langage: PHP in version 5+ for object-oriented programming
